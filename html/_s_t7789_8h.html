Detailed Description

Driver program for ST7789 LCD Display.

Author
Michael Burmeister
Date
December 7, 2021
Version
1.0

Because the driver memory is 240x320 and the screen is 135x240 there are blank areas that do not show up on the screen IE in non display area. Need to offset screen memory by 40 pixels and the row by 52/53 pixels Driver is to slow for anything above 60Mhz.
