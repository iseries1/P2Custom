bme68x_set_regs

int8_t bme68x_set_regs(const uint8_t reg_addr, const uint8_t *reg_data, uint32_t len, struct bme68x_dev *dev)
This API writes the given data to the register address of the sensor
Parameters
[in]reg_addr: Register addresses to where the data is to be written
[in]reg_data: Pointer to data buffer which is to be written in the reg_addr of sensor.
[in]len: No of bytes of data to write
[in,out]dev: Structure instance of bme68x_dev
Returns
Result of API execution status
Return values
0-> Success
<0 -> Fail

int8_t bme68x_get_regs(uint8_t reg_addr, uint8_t *reg_data, uint32_t len, struct bme68x_dev *dev)

[out]reg_data: Pointer to data buffer to store the read data.

[in,out]dev: Structure instance of bme68x_dev.
