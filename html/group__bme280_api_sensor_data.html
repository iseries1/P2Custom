bme280_get_sensor_data

int8_t bme280_get_sensor_data(uint8_t sensor_comp, struct bme280_data *comp_data, struct bme280_dev *dev);
This API reads the pressure, temperature and humidity data from the sensor, compensates the data and store it in the bme280_data structure instance passed by the user.
Parameters
[in]sensor_comp: Variable which selects which data to be read from the sensor.
sensor_comp Macros
1 BME280_PRESS
2 BME280_TEMP
4 BME280_HUM
7 BME280_ALL
Parameters
[out]comp_data: Structure instance of bme280_data.
[in]dev: Structure instance of bme280_dev.
Returns
Result of API execution status
Return values
0-> Success.
>0 -> Warning.
<0 -> Fail.

bme280_parse_sensor_data

void bme280_parse_sensor_data(const uint8_t *reg_data, struct bme280_uncomp_data *uncomp_data);
This API is used to parse the pressure, temperature and humidity data and store it in the bme280_uncomp_data structure instance.
Parameters
[in]reg_data: Contains register data which needs to be parsed
[out]uncomp_data: Contains the uncompensated pressure, temperature and humidity data.

bme280_compensate_data

int8_t bme280_compensate_data(uint8_t sensor_comp,
const struct bme280_uncomp_data *uncomp_data,
struct bme280_data *comp_data,
struct bme280_calib_data *calib_data);
This API is used to compensate the pressure and/or temperature and/or humidity data according to the component selected by the user.
Parameters
[in]sensor_comp: Used to select pressure and/or temperature and/or humidity.
[in]uncomp_data: Contains the uncompensated pressure, temperature and humidity data.
[out]comp_data: Contains the compensated pressure and/or temperature and/or humidity data.
[in]calib_data: Pointer to the calibration data structure.
Returns
Result of API execution status.
Return values
0-> Success.
>0 -> Warning.
<0 -> Fail.
